# Data control and organisation 
## Data control
### Garbage in, garbage out
Errors that can occur during input of data include:
- **Data source error**: Incorrect data is provided by data source.
- **Transcription error**: Misread, misheard or mistyped data
- **Transposition error**: Incorrect placement of the data.
When dumb data is input to analysis, the output information will be dumb. This is called "garbage in, garbage out"
## Data Validation
This is part of data control. While it cannot guarantee that the input data is absolutely accurate, it ensures such data is reasonable and matches set criteria.

Table: List of data validation methods

| Type              | Function                                                                           |
| ----------------- | ---------------------------------------------------------------------------------- |
| Presence Check    | Ensures there must be data in the required fields                                  |
| Fixed Value Check | Ensures that the data can only be selected from a list of specific values          |
| Type Check        | Ensures that the data must be of a certain type                                    |
| Range Check       | Ensures that the data must be within a certain range                               |
| Length Check      | Ensures that the length of the data must match certain criteria                    |
| Format Check      | Ensures that the data must be in a specific format                                 |
| Consistency Check | Ensures that the data must be reasonable when compared to the data in other fields |
| Uniqueness Check  | The data must be unique for the field                                              |
>[!info] UI designs that help facilitate data validation
>Drop-down lists, radio buttons and checkboxes are fixed value checks.
>Date Picker is both a type check and a format check.
### Check Digit
>[!info] Check digit of the HKID
>Steps of validating an HKID number
>1. If an HKID number starts with only one English letter, a whitespace is added to the beginning.
>2. Convert each English letter into a number, with A-Z being 10-35. Whitespace is converted to 36.
>3. Each digit has a weight that descends with the first digit having a weight of 9 and the ending digit having a weight of 1.
>4. Divide the weighted sum by 11. The HKID number is valid if there is no remainder.
